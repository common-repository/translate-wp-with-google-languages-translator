Simple and powerful Google Translator plugin. Use it with a shortcode or with a widget, and make your website multilingual and accessible to everybody on the internet.

== Description ==

Using the power of Google, you can translate your WordPress powered site with no effort, and make it multilingual to everybody on the internet. 

**Features**

* 100% free plugin
* Show/hide Google top banner
* Flags combined into one file
* Google Analytics integration
* Translates the site on the fly
* Valid XHTML

You might want take a brief look at this premium translation plugins guide: [https://wordpress.translate.guide](https://wordpress.translate.guide)

== Installation ==

1. In the WordPress administration back end tool, go to: Plugins - Add New, and click on the Upload Plugin button.
2. Select the translate-wordpress-google-languages-translator.zip file. Install it and activate it.
3. To adjust the settings, go to: Settings - Google Languages Translator.
4. Use it with a shortcode or with a widget. Check the Usage tab for more details.
